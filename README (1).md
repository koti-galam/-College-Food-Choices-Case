# Comprehensive Analysis and Dietary Strategies with Tableau: A College Food Choices Case Study

## 📌 Overview

This project analyzes college students' dietary habits, lifestyle
choices, and academic performance using Tableau dashboards.

## 🎯 Objectives

-   Analyze dietary habits and lifestyle patterns.
-   Study the relationship between food choices and GPA.
-   Develop interactive dashboards.
-   Generate actionable insights for healthier living.

## 🛠️ Technologies Used

-   Tableau Desktop
-   Python (Pandas, NumPy)
-   Microsoft Excel
-   CSV Dataset

## 📊 Features

-   Interactive Dashboards
-   KPI Cards
-   Dynamic Filters
-   Story Dashboards
-   Trend Analysis

## 📂 Dataset

The dataset includes breakfast habits, exercise frequency, comfort food
preferences, calorie awareness, fast food consumption, fruit & vegetable
intake, and GPA.

## 📈 Dashboard Insights

The dashboard contains the following visualizations:

1.  KPI Cards (Count of GPA, Average GPA, Average Weight, Average Grade
    Level)
2.  Breakfast Consumption (Horizontal Bar Chart)
3.  Health vs Greek Food Consumption (Clustered Bar Chart)
4.  Vitamins Consumption (Line Chart)
5.  Weight vs Exercise (Heat Map)
6.  Eating Out vs Cooking Frequency (Dual Line Chart)
7.  Comfort Food Reasons (Packed Bubble Chart)

These visualizations provide insights into dietary habits, exercise
patterns, nutrition awareness, and their relationship with academic
performance.
